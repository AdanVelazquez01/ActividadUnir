# Blog Saludable

Proyecto desarrollado como actividad práctica sobre conceptos básicos de HTML y CSS.

## Estructura del Proyecto

El proyecto final compilado tiene la siguiente estructura:

```text
├── login.html           (Página de inicio de sesión)
├── dashboard.html       (Dashboard - Página principal)
├── caminar.html         (Página de entrada de blog sobre Caminar)
├── comer-bien.html      (Página de entrada de blog sobre Comer Bien)
├── ejercicio.html       (Página de entrada de blog sobre Ejercicio en casa)
├── assets/
│   ├── css/
│   │   └── style.css    (Estilos globales del sitio)
│   ├── js/
│   │   └── login.js     (Lógica del inicio de sesión)
│   └── img/
│       ├── caminar.png
│       ├── comer-bien.png
│       └── ejercicio.png
└── README.md            (Archivo de descripción del proyecto)
```

## Características

- Sitio web estático rápido e independiente.
- Rutas relativas para fácil visualización de forma local.
- Estilos personalizados modernos.
- Simulación de login (`admin` / `1234`).
